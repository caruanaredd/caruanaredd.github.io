What is .NET?

  • Network Enabled Technology

    Capable of translating various programming languages (C#, F#, VB.NET)
    into functional applications.

  • Shared Libraries

    Developers are not required to write the foundational code from scratch
    every time they start a new project.

    Packed with tens of thousands of utility functions that will speed up production.

  • Free, Cross-Platform, Open-Source

    Any developer can actively contribute to the framework and help it grow.

What is C#?

A general-purpose type-safe programming language created by Microsoft.

  • Common Language Runtime

    Projects can be simplistic in structure, or consist of numerous
    classes containing various relevant functions.

  • Heavily integrated with the .NET Framework

    We don't need to do any heavy lifting to get our projects to work!

Why Learn C#?

  • Popular and versatile coding language

    Used to create interactive websites, mobile apps, digital games, as well as augmented and virtual Reality.

    Pokémon Go and the Stack Overflow website
    were built using versions of C# (Unity and ASP.NET respectively).

  • Designed with a shallow learning curve.

    Easy to learn and get creating, but takes some time and practice to master.

    Much easier than Java and C++, extremely flexible and scalable by comparison.

  • Programming teaches you how to think.

    Improve your problem-solving skills, learn how to approach a project,
    how to stimulate your mood and creativity.

The biggest weakness of C#

All code needs to be compiled before it can run, making development slow.
